Last Definition: Virtual 2D/3D file of a last, i.e. a mechanical form with a shape of a human foot,
which is used to design and manufacture a shoe on it.
Availability: There are in total ca. 500 lasts which exist in different sizes and forms
(e.g. low cut, high cut, etc.). Ca. 50 new lasts are created per year.
.stl files are existing for each last in the RCS (database) and those files can automatically be transferred into .obj files.
At the moment, .stl files can be transferred into .3DM files,
but it is not an automated process yet (the automated process is curretly being built)		 Create Tools:
- TAPE
- Rhino
- Clo
File Format:
.stl (file for production)
.obj (file to upload in TAPE)
.3DM (file to upload in Rhino)
.dxf (2D file of the bottom gauge
/ has the form of a normal insole)		 The last database RSC (= all existing .stl files), is currently owned be the Last Department (Jürgern Schacher).
ACT has ownership on the 3D files (.obj, and .3DM).
Going forward, the governance should lie with the Last Department (in particular on controlling access)		 MVP Digital Libraries Draft
